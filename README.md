# go-metabase
Metabase API Client for Golang

## Installation
```
go get github.com/kucuny/go-metabase/metabase
```

## Usage
```go
import "github.com/kucuny/go-metabase/metabase"
```

## Features

- [ ] Activity

  - [ ] Get recent activity.
  - [ ] Get current user's recent activity
